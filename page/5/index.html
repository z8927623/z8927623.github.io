使用clang语法消除编译警告

发表于 2015-11-24   |  

自己总结了一些常用的clang语法用于消除Xcode警告

用法

Warning Message
-Warc-performSelector-leaks performSelector可能会造成内存泄露因为无法知道具体的selector
-Wunreachable-code" 无法执行到的代码
-Wunused-function 没有使用的函数
-Wdeprecated-implementations 标记为已经废弃的方法
-Wdeprecated-declarations 消除使用废弃方法警告
-Wreceiver-is-weak和-Warc-repeated-use-of-weak 接受者是weak属性
-Wundeclared-selector 未定义的selector
-Wgnu 未知
-Wreturn-type control reaches end of non-void function
-Wimplicit-function-declaration implicit declaration of function
-Wconversion 消除精度丢失warning
